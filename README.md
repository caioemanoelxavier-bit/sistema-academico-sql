# 🎓 Sistema Acadêmico - Parte 1 (Modelagem)

Projeto desenvolvido para a disciplina de Banco de Dados (ADS - UNICID). 
O objetivo é estruturar a base de dados de um ERP educacional.

## 📂 Estrutura do Projeto
- **Módulo Acadêmico**: Gestão de alunos, disciplinas, notas e faltas.
- **Módulo Financeiro**: Controle de contratos e mensalidades.
- **Módulo RH**: Gestão de funcionários e corpo docente.

## 🛠️ Tecnologias Utilizadas
- Banco de Dados Relacional (MySQL/MariaDB)
- Ferramenta de Modelagem
- VS Code

## 📝 Como executar
Os scripts de criação das tabelas estão localizados na pasta `scripts/`.
